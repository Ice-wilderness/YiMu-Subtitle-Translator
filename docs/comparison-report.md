# 译幕 vs VideoLingo 对比测试报告

## 测试条件

| 项目 | 内容 |
|------|------|
| 测试视频 | Anthropic Just Dropped the New Blueprint for Long-Running AI Agents（17 分钟） |
| 翻译模型 | 两者使用同一个翻译模型（控制变量） |
| 测试时间 | 2026-04-01 |
| 测试机器 | 同一台 Windows 机器，CUDA GPU |

---

## 一、速度对比

两者调用同一个翻译模型，仅对比翻译流程（不含转录）。

### 译幕

| 阶段 | 耗时 |
|------|------|
| 内容分析（生成 52 个术语） | ~2 分钟 |
| 4 批并行翻译 | 48 秒 |
| 质量检测 + AI 修复（2 条诊断） | 7 秒 |
| **翻译流程合计** | **约 3 分钟** |

### VideoLingo

| 阶段 | 耗时 |
|------|------|
| 字幕断句 + 清洗 | ~1 分钟 |
| 术语表 + 摘要 | ~14 秒 |
| 翻译（忠实度 + 表达力两轮） | ~3 分 38 秒 |
| 字幕裁剪 + 对齐 + 输出 | ~5 分 40 秒 |
| **翻译流程合计** | **约 10 分钟** |

### 速度结论

译幕翻译速度约为 VideoLingo 的 **3 倍**。

## 二、翻译质量对比

### 2.1 专有名词准确性（ASR 矫正的价值）

VideoLingo 没有 ASR 矫正环节，语音识别错误直接进入翻译。译幕在翻译前通过 AI 矫正修正了这些错误。

| 原文（正确） | VideoLingo 翻译 | 译幕翻译 |
|-------------|----------------|---------|
| Claude Agent SDK | ❌ **Cloud** Agent SDK（出现 3 处） | ✅ **Claude** 智能体 SDK |
| Sonnet 4.5 | ❌ **Solid** 4.5 | ✅ **Sonnet** 4.5 |
| handing off context cleanly | ❌ handing off **contacts** cleanly | ✅ 干净利落地**交接上下文** |

这类错误对观众影响很大：看到 "Cloud Agent SDK" 会直接影响对内容的理解。

### 2.2 术语一致性

| 项目 | 译幕 | VideoLingo |
|------|------|-----------|
| 自动术语表 | ✅ 52 个术语自动生成 | 有术语提取，数量未知 |
| harness 翻译 | 全文统一为"控制框架" | 混用"harness 控制层"、"harness（控制与编排层）"等多种写法 |

### 2.3 翻译风格

测试视频为一期 AI 技术解读（*Anthropic Just Dropped the New Blueprint for Long-Running AI Agents*），内容涉及 GAN 架构、上下文焦虑、评估智能体设计等专业话题。译幕会根据原文内容自动适配翻译风格。

以下为同一句话在两者中的实际翻译差异：

| 原文 | VideoLingo | 译幕 |
|------|-----------|------|
| it's likely going to praise it | "它十有八九会把自己夸得挺好" | "它很可能会给自己好评" |
| it costs $100 to create | "这玩意儿也就花了100美元" | "它的制作成本只有100美元" |
| I'm sure they're delighted | "他们肯定高兴坏了" | "他们肯定乐见其成" |
| they approved the work anyway | "但最后还是把这个结果给通过了" | "最后还是通过了这份工作" |

| 维度 | VideoLingo | 译幕 |
|------|-----------|------|
| 本次测试风格 | 口语化 | 书面化 |
| 字幕条数 | 413 条（2-4 秒/条） | 323 条（4-8 秒/条） |

### 2.4 翻译准确性

两者均使用同一翻译模型，纯翻译准确性无明显差距。差异主要来自 ASR 矫正和术语管理。

## 四、核心结论

1. **译幕的核心优势是翻译准确性**：ASR 矫正环节让专有名词翻译显著优于 VideoLingo（Claude vs Cloud、Sonnet vs Solid）
2. **译幕速度更快**：并行翻译 + 轻量检测修复，纯翻译速度约为 VideoLingo 的 3 倍
3. **译幕有完整的质量保障链**：翻译后的去重、检测、AI 修复是 VideoLingo 没有的
4. **VideoLingo 功能更全**：视频烧录、多语言、免费翻译、批量处理等是译幕目前不具备的
5. **两者定位不同**：VideoLingo 是全流程工具，译幕专注翻译质量

**适用场景建议：**
- 需要快速批量处理、多语言、视频烧录 → VideoLingo
- 对英→中翻译质量有高要求（字幕组、技术内容、专业场景）→ 译幕
