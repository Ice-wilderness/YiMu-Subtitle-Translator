🎬 欢迎使用译幕

5 分钟快速上手，跟着引导完成第一个字幕翻译项目

⚙
首先：配置软件设置

点击右上角 ⚙ 软件设置 按钮，填写 AI 模型的 API 信息。不配置则无法使用矫正和翻译功能。

  • 分析模型：用于识别字幕中的误识别词，已测试：Gemini 2.5 Flash / Pro
  • 翻译模型：用于将字幕翻译成中文，已测试：GPT-4o / GPT-5.1
  • ⚠ 目前仅针对英语→中文翻译进行了测试，其他语言对未验证
  • 两个模型可以配置不同的 API Key 和 Base URL，也可以填同一个
  • 配置保存后永久生效，下次打开无需重新填写
1
导入文件

将文件拖入上传区，或点击选择文件。支持两种方式：① 直接上传已有的 .srt 字幕文件；② 上传音视频文件（MP4、MKV、MP3 等），由本地 Whisper 模型自动转录生成字幕。

  • 上传 SRT 文件后会直接显示字幕统计，可继续下一步
  • 上传音视频后会展开转录设置，选择模型和设备后点「开始转录」
  • 文件数据仅保存在浏览器内存中，刷新页面数据会清空
2
短句合并

语音识别经常把短词单独切成一行字幕（如 "the"、"and"、"的"、"了"），此步骤自动将这些碎片合并到上下文句子里。

  • 「最大单词数」滑块控制多少个词以内算作短句，默认 3 个
  • 点击「执行合并」后不可撤销，建议先确认参数
  • 合并后会显示合并了多少条字幕
3
AI 矫正 & 翻译

分两个阶段：① 用 AI 分析语音识别的错别字/误识别词，② 对矫正后的文本进行中文翻译。推荐使用「一键翻译」，自动完成翻译+质量检测+AI修复。

  • 重要：填写视频背景（标题和内容简介），AI 矫正效果更好
  • 一键翻译（推荐）：自动执行内容分析→并行翻译→质量检测→AI修复
  • 也可以手动分步操作：先「分析」矫正，再「最终翻译」
4
导出

可选调换双语上下顺序，预览最终字幕，下载 SRT 文件导入剪辑软件。

  • 位置调换是可选操作，选择后立即生效
  • 顶部导航每个步骤变绿表示已完成
