# Changelog

## v2.1 (2026-03-31)

### 修复
- **JS 空元素报错**：修复 `transBtn` 元素不存在导致 JS 崩溃，造成一键翻译、提交修正等按钮全部失效的问题

### 新增
- **去重修复（阶段 2.5）**：翻译完成后、质量检测前新增一道纯程序去重步骤。检测相邻条目是否存在"N+1 条译文以 N 条译文为前缀"的重复拼接，直接裁掉重复部分，无需调用 AI，运行耗时 < 1ms。新增配置项 `dedup_min_overlap`（默认 8 字）控制触发阈值
- **重复合并防护**：AI 修复阶段新增 `already_merged` 集合，任何参与过合并的条目（被删或被修改）均不再参与二次合并，防止相邻两条同时被检测时产生重复翻译

### 变更 — AI 智能修复逻辑重构
- **AI 不再生成译文内容**：原方案 AI 直接输出合并后的 `new_translation`，导致译文重复拼接（幻觉）；新方案 AI 只返回合并方向（`merge_up` / `merge_down` / `skip`），译文由翻译 API 重新生成
- **工程化合并**：软件负责拼接原文文本和时间轴，拼接后原文重新调用翻译 API 翻译，彻底消除 AI 幻觉导致的译文污染
- **去除 `retranslate` action**：翻译对齐有三重匹配保障，错位概率极低，不再需要单条重译
- **新增时间间隔拦截**：merge 执行前检查两条字幕之间的时间间隔，超过阈值（默认 500ms）强制跳过，防止跨说话人合并
- **新增配置项** `merge_max_gap`（默认 0.5s）：可在 config.json 中调整间隔阈值
- **新增修复详情日志**：每条修复操作记录修改前/修改后内容、时间轴、被拦截原因，方便后台追踪

## v2.0 (2026-03-23)

### 新增
- **一键翻译**：点击一个按钮，自动串联执行 内容分析→并行翻译→质量检测→AI智能修复，全程无需手动干预
- **内容分析**：翻译前自动取样字幕，AI 生成领域识别 + 术语表，注入每批翻译 prompt，保证跨批次术语一致性
- **并行翻译**：多批次并行发送 API 请求（默认 3 批并发），大幅提升翻译速度
- **AI 智能修复**：翻译后自动质量检测，将问题条目+上下文发给 AI 诊断（合并/重译/确认无问题），自动应用修复
- **新增配置项**：`parallel_batches`（并行批次数，默认 3）
- **新增 API**：`POST /api/oneclick`（一键翻译）、`GET /api/oneclick/status/<id>`（进度轮询，含阶段信息）

### 变更
- **Tab 从 7 个精简为 4 个**：导入→短句合并→AI矫正&翻译→导出
  - 原 Tab 4（质量检测）和 Tab 5（字幕修复）整合进一键翻译自动流程，不再需要独立 Tab
  - 原 Tab 6（位置调换）+ 原 Tab 7（下载导出）合并为「导出」
- 普通翻译也改为并行执行（复用 `parallel_batches` 配置）
- 翻译 prompt 支持术语表注入（`glossary_hint` 参数）

### 修复
- SRT 解析器多行文本保留 `\n` 而非用空格合并，修复双语字幕中英混在同一行的问题

## v1.0 (2026-03-10)

### 功能
- **Tab 1 导入文件**：支持 SRT 文件上传和音视频文件本地 Whisper 转录（双引擎：CLI 优先，Python 库备选）
- **Tab 2 短句合并**：按语言类型（CJK/Latin）自动合并碎片字幕，可调节合并阈值
- **Tab 3 AI 矫正 & 翻译**：
  - AI 语音识别矫正：分析字幕中的误识别词（如 "deep sea" → "DeepSeek"），支持多轮分析
  - 字幕翻译：分批调用翻译模型 API，逐行对齐输出双语 SRT
- **Tab 4 质量检测**：检测中文字符过少（漏翻）和持续时间过短的问题字幕
- **Tab 5 字幕修复**：逐条决策合并问题字幕，内置浮窗字幕编辑器（Ctrl+F 打开，支持搜索替换）
- **Tab 6 位置调换**：双语字幕中英上下顺序互换
- **Tab 7 下载导出**：预览最终字幕，下载 SRT 文件

### 技术架构
- 纯前端 SPA + 轻量 Flask 后端（waitress 9999 端口）
- 数据全部在浏览器内存，无数据库
- 两套独立 API 配置（分析模型 / 翻译模型）
- Whisper 转录双引擎：`faster-whisper-xxl.exe` CLI 优先，`faster_whisper` Python 库备选
- VAD 自动降级：过滤 >75% 音频时自动关闭 VAD 重试
